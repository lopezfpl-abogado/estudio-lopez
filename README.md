# Estudio Jurídico Juan Fernando López

Sitio web del estudio con portal del cliente conectado a Google Sheets.

## Tecnologías

- React 18 + Vite
- Tailwind CSS
- Lucide React (iconos)

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue

Conectado a Vercel — cada push a `main` despliega automáticamente.
